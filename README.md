# Loan Approval Prediction System

## 📌 Overview
This project predicts loan approval status using machine learning based on applicant financial and demographic data.

## ⚙️ Technologies Used
- Python
- Scikit-learn
- Pandas, NumPy, Matplotlib, Seaborn

## 🔍 Features
- Data preprocessing and feature engineering
- Model training using Logistic Regression, KNN, Naive Bayes
- Model evaluation using accuracy, precision, recall, F1-score

## 📊 Results
- Compared multiple models (Logistic Regression, KNN, Naive Bayes) under different parameter settings.
- Observed varying performance across models depending on feature selection and preprocessing.
- Evaluated using accuracy, precision, recall, and F1-score to ensure balanced performance.

- data/ → dataset
- notebook/ → Jupyter notebook
